# **DeepRacer-For-Dummies - Starting/Stopping Training**

1. Start the training:

    ```terminal
    cd ~/git/deepracer-for-dummies/scripts/training && \
        ./start.sh
    ```

2. Stop the training:

    ```terminal
    cd ~/git/deepracer-for-dummies/scripts/training && \
        ./stop.sh
    ```
