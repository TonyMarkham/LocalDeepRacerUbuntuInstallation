# **DeepRacer-For-Dummies - Starting/Stopping Training Evaluation**

1. Start evaluating the training:

    ```terminal
    cd ~/git/deepracer-for-dummies/scripts/evaluation && \
        ./start.sh
    ```

2. Stop evaluating the training:

    ```terminal
    cd ~/git/deepracer-for-dummies/scripts/evaluation && \
        ./stop.sh
    ```
